## Flower 项目简介
    一个玫瑰花商城项目，商城项目包含了较复杂和繁琐的逻辑和交互，所以选择做一个商城项目
* 首页选项卡有`轮播`、`推荐商品`、`商品分类`、`商品概览`，
* 第二个选项卡是`客服聊天`功能
* 第三个选项卡是`购物车`功能
* 第四个选项卡是`个人中心`功能
* 项目结合了 `Ionic + Angular` 混合开发实现多平台部署
* `前端(Ionic+Angular)` + `后端(PHP)` + `UI` + `数据库(Mysql)` + `爬虫(NodeJs)` 全是个人 `独立开发`

## 项目部署
* 前端页面实现：`Ionic + Angular`
* 后端：`PHP`
* 数据库： `Mysql`
* 服务器：`BlueHost`
